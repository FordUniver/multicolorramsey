# Mathlib Contributions

This directory contains code that has been or will be contributed to Mathlib.
Each subdirectory represents a separate contribution with its own metadata.

**The main MulticolorRamsey codebase imports from this directory.** When a contribution
is merged into Mathlib, we update the imports to use the Mathlib version instead.

## Structure

Each contribution has:
- `ContributionName.lean` - The actual code used by the project
- `metadata.yaml` - Tracking information (status, PR number, description, etc.)

See the auto-generated overview at: https://forduniver.github.io/multicolorramsey/mathlib-contributions.html
